# Easy Shopping Via Mobile

Projeto desenvolvido para treinar HTML, CSS e responsividade, criando uma landing page moderna com foco em adaptação para dispositivos móveis.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Media Queries
- Flexbox
- Google Fonts

## 📱 Responsividade

O projeto foi desenvolvido com responsividade para diferentes tamanhos de tela, incluindo:

- Desktop
- Tablet
- Mobile

## 🎨 Funcionalidades

- Layout moderno
- Botão com efeitos hover e active
- Navegação estilizada
- Responsividade com Media Queries
- Estrutura organizada com sections e containers

## 📚 O que pratiquei

Neste projeto pratiquei:

- Posicionamento com CSS
- Responsividade
- Uso de `display inline-block`
- `position: absolute` e `relative`
- Media Queries
- Estilização de botões
- Organização de layout
- Tipografia com Google Fonts
